# Chromezy Next.js SSR

Chromezy Next.js SSR is a modern web application built with Next.js, Tailwind CSS for styling, and a modular architecture. The project includes reusable components, containers for page sections, and hooks for enhanced interactivity. Animations are powered by framer-motion for smooth and interactive UI transitions.

## How to Run

1. **Install dependencies:**
   npm install

2. **Start development server:**
   npm run dev

The app will be available at [http://localhost:3000]

3. **Using Docker:**
   If you want to run the app in a Docker container:
   docker-compose up --build
